# 🔑 Comment Obtenir une Clé Groq (Gratuit)

## ⚡ Pourquoi une clé Groq ?

La clé Groq permet au dialogue IA de fonctionner. Groq offre **gratuitement** :
- ✅ **30 requêtes/jour** (suffisant pour une utilisation normale)
- ✅ **Modèles puissants** (Mixtral, LLama 2)
- ✅ **Instant** - aucune attente
- ✅ **Aucune carte bancaire** nécessaire

---

## 📋 Étapes pour Obtenir Ta Clé

### 1️⃣ Va sur Groq Console
Ouvre : **https://console.groq.com**

### 2️⃣ Crée un Compte
- Clique "Sign Up"
- Utilise ton email
- Valide ton email (regarde dans les spams si nécessaire)
- Crée un mot de passe

### 3️⃣ Génère une API Key
- Va dans l'onglet **"API Keys"** (à gauche)
- Clique sur **"Create API Key"**
- Donne-lui un nom (exemple : "Lina's App")
- Clique **"Generate"**
- Une clé apparaît : `gsk_...`

### 4️⃣ Copie la Clé
- Clique sur l'icône "Copier" 📋
- La clé est dans ton presse-papiers
- ⚠️ **Ne la ferme pas !** Tu auras besoin de la copier

### 5️⃣ Mets la Clé dans l'App
**Sur ton téléphone/ordinateur :**
1. Ouvre l'app (https://got-dragon.github.io/Lina_English)
2. Clique sur ⚙️ **Paramètres** (en haut à droite)
3. Dans "🔑 Clé Groq" → **Colle ta clé** (Ctrl+V ou Cmd+V)
4. Clique **"💾 Sauvegarder la clé"**
5. ✅ C'est fait !

---

## 🎯 Vérifier que ça Marche

1. Va dans l'onglet **💬 Dialogue**
2. Sélectionne un mode (par exemple "Discussion libre")
3. Tape une question simple : "Hello, how are you?"
4. Clique **Envoyer**
5. L'IA doit répondre en 2-3 secondes ✅

Si ça ne marche pas :
- ❌ "⚠️ Ajoute ta clé Groq" → La clé n'est pas sauvegardée. Recommence étape 5️⃣
- ❌ "❌ Erreur API" → Vérifie que la clé commence par `gsk_`
- ❌ "❌ Erreur réseau" → Vérifie ta connexion Internet

## 🆘 Troubleshooting

### "La clé ne fonctionne pas"
- Vérifie qu'elle commence par `gsk_`
- Copie-la sans espaces avant/après
- Rafraîchis la page (F5)

### "J'ai perdu ma clé"
- Aucun problème ! Va sur https://console.groq.com
- Clique sur ta clé → "Regenerate"
- Tu auras une nouvelle clé
- Sauvegarde-la dans l'app

### "Je veux changer de clé"
- Paramètres ⚙️ → Collez la nouvelle clé
- Clique "Sauvegarder"
- Voilà !

### "J'ai atteint la limite de 30/jour"
- Attends le jour suivant (minuit UTC)
- Aucune charge supplémentaire
- La limite se réinitialise automatiquement
